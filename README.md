# 🎓 Sistema de Acompanhamento Acadêmico

Projeto desenvolvido para a disciplina Banco de Dados da Universidade de Brasília (UnB).

## Objetivo

Desenvolver um sistema capaz de acompanhar a situação acadêmica dos estudantes durante o semestre letivo, permitindo o gerenciamento de notas, frequência, avaliações e notificações de desempenho.

O sistema busca auxiliar alunos e professores no acompanhamento contínuo do progresso acadêmico.

## Funcionalidades

### Aluno

- Consultar disciplinas matriculadas
- Visualizar notas
- Acompanhar frequência
- Receber notificações
- Consultar eventos acadêmicos
- Acompanhar metas de estudo

### Professor

- Registrar avaliações
- Lançar notas
- Registrar frequência
- Disponibilizar materiais
- Criar eventos acadêmicos
